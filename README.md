# img_processing

第一回(終了): カラー画像をグレイスケール画像に変換 → grayscale

第三回(終了): グレイスケール画像の縮小・拡大・回転 → scaling_and_rotation

第四回(終了): 濃度度数分布表とヒストグラムの出力、ヒストグラム平滑化 → histogram

第五回: ノイズ除去と鮮鋭化 → denoise_and_sharp

※ 実行時、第一引数に変換したい画像のパスを指定する。
